▪️サービス概要
高校野球が好きな人が集まるサービス。

◾️このサービスへの思い、作りたい理由
高校までずっと野球をしてて、野球を見るのが好きなんですけど、一緒に見に行ったり語る人が少ないので、集めれる場を作れればもっと楽しくなるかなと思いました。

▪️ユーザー層について
まずは高校野球が好きな人。おいおいは野球好き全般になればいいかなと思います。

▪️サービスの利用イメージ
投稿機能を作り、意見や情報交換日常などを発信できるサービス。あとは、高校野球のトーナメント表欄を作り、試合日程などを確認できる。高校ごとに強さランキングや匿名の討論欄も検討中です。

◾️ユーザーの獲得について
まずは野球が好きな人が集まるので、スムーズに会話できると思う。投稿機能を使わなくてもトナメ表で試合日程や勝ち負け、試合時間どこで試合するかなどをすぐ確認できる。

◾️サービスの差別化ポイント、推しポイント
高校野球に特化して、試合日程や勝ち負けをすぐわかりやすく、野球好きと繋がりやすいのはもちろん、討論欄も作れればより楽しめると思う。

◾️機能候補
ユーザー登録機能。本文投稿機能。投稿いいね機能。投稿検索機能。投稿にコメント機能。トーナメント表欄
本リリース　個人チャット機能

▪️機能の実装方針予定
トナメ表欄に球場情報も入れればと思っており、google mapsと連携したい。
